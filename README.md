# Required
```bash
sudo apt install grep
```

# Installation
```bash
mv ~/.config/nvim ~/.config/nvim.back 

git clone https://github.com/khtodod/nvim.git ~/.config/nvim

nvim

:PlugInstall
```

# Features
```js
- Format on save
- Auto install language server
- Auto close // <> "" ''
```

# Shortcuts
```js
<Leader> = " " // space
<C> = ctrl

gcc // code comment out
<C>e // scroll to bottom
<C>y // scroll to top
<Leader>ff // find files in the project.
<Leader>fg // search for text inside files (like grep).
<Leader>fb // List open buffers.
<Leader>fh // search through help tags.
<Leader>fr // show recent files
<C>\ // toggle terminal
<Leader>sv // split window vertically
<Leader>sh // split window horizontally
<Shift>h // go to start of line
<Shift>l // go to end of line
```
